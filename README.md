# 携帯電話番号判定プログラム (初級理解度チェック 課題3)
## 概要
正規表現を使用して、ハイフン付きの国内携帯電話番号の有効性をチェックするプログラムを作成しました。

以下のパターンを有効な携帯電話番号とし、その条件に合致するかどうかを判定しました。
